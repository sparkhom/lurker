# Caint

Self-hosted IRC client. Server holds a persistent IRC connection; a Vue web UI connects over WebSocket to view and interact with channels.

## Stack
- Express + `irc-framework` + `ws` + better-sqlite3 (server)
- Vue 3 + Vite + Pinia (client)

## Development
```
npm run install:all
cp .env.example .env   # edit SESSION_SECRET; defaults assume the local hostname below
npm run dev
```

Auth is passkey-only. On first launch the UI prompts you to create your account + first passkey.

The dev hostname defaults to `irc.local.bradroot.me` over HTTPS (WebAuthn requires HTTPS for any non-localhost RP). Map it to 127.0.0.1 in your hosts file:

```
127.0.0.1   irc.local.bradroot.me
```

Vite serves the client at https://irc.local.bradroot.me:5173 (with a locally-trusted cert generated by `vite-plugin-mkcert`) and proxies `/api` and `/ws` to the Express server on `PORT` (default 8010).

## Production
```
npm install
npm run client:build
npm start
```
